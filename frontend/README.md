# ELS Bank Frontend

Improved Vue 3 + Pinia + Bootstrap frontend for the ELS Bank backend.

## Features

- Real authentication flow
- JWT storage
- Route guards
- Login required
- Customer and employee roles
- Real backend account loading
- Transactions from backend
- Transfer money
- Registration flow
- Employee approval dashboard
- Bootstrap styling
- Axios API layer
- Vue Router
- Pinia state management

## Start

```bash
npm install
npm run dev
```

Backend must run on:

http://localhost:8080
